Privacy policy


As used herein, the term "personal information" means information that specifically identifies an individual (such as a name, address, telephone number, mobile number, e-mail address, or credit card number), or information about that individual that is directly linked to personally identifiable information.
Personal information is the subject of this privacy policy.

The application "IQ'biki" will not try compromise the security of user personal information.
This application don't collect and don't request any information about the personal data of user as email, phone numbers, user location or any like.

1. The phone's memory is used by application to store user-made settings and the results of gaming sessions.
Saved data can be easily removed from the device memory common to all applications way: [Settings] → [Applications] → [IQ'biki] →[ Delete data, Clear the cache].
2. Access to the user's contacts is required by the user chosen communicator in case user himself decides to share the results of his gaming session with any of his contacts.

Политика конфиденциальности

В используемом здесь значении термин «личная информация» означает информацию, которая конкретно идентифицирует отдельного человека (например: имя, адрес, номер телефона, номер мобильного телефона, адрес электронной почты или номер кредитной карты) или другую информацию об этом лице, которая непосредственно связана с личной информацией.
Личная информация является предметом данной политики конфиденциальности.

Приложение «IQ'biki» не будет пытаться нарушить безопасность личной информации пользователя.
Это приложение не собирает и не запрашивает какую-либо информацию о личных данных пользователя в виде, например, электронной почты, номеров телефонов, местонахождения пользователя и т.п.

1. Память телефона используется приложением для хранения пользовательских настроек и результатов игровых сессий. Сохраненные данные могут быть легко удалены из памяти устройства, общим для всех приложений способом: [Настройки] → [Приложения] → [IQ'biki] → [Удалить данные, Очистить кеш].
2. Доступ к контактам пользователя запрашивается выбранным пользователем коммуникатором в том случае, если пользователь сам решит поделиться результатами своей игровой сессии с каким-либо из своих контактов.
